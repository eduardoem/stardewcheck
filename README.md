
# Checklist de Peces - Stardew Valley

## Características
- PWA instalable en Android
- Funciona offline
- Guarda progreso automáticamente

## Publicación en GitHub Pages
1. Subir el contenido al repositorio.
2. Activar GitHub Pages desde Settings > Pages.
3. Seleccionar la rama principal.
